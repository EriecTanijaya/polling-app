## How

### Local
- start service with [xampp](https://www.apachefriends.org/index.html)
- Clone this repo
- Import the phppoll.sql to the phpmyadmin

### Online Host
- Clone this repo
- Import the phppoll.sql to the phpmyadmin
- Fill the database config in
  - functions.php
  - auth.php
  - profile.php
  - register_action.php
